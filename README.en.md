# Urestic

Urestic is being rebuilt as an easy-use web tool for restic.

It is no longer a central backup executor or a traditional GUI for restic. Each server runs its own restic backup script locally. Urestic helps generate scripts, manage repository settings, query snapshots, analyze backup health, and configure result notifications.

The Chinese README covers usage. `AGENTS.md` is the source of truth for the current product direction.
